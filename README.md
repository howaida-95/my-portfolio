# my-portfolio

Personal portfolio built with [Next.js](https://nextjs.org) (App Router), Tailwind CSS, and shadcn-style UI.

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Environment

Copy `.env.example` to `.env.local` and set SMTP variables for the contact form. Do not commit `.env.local`.

## Deploy

Deploy on [Vercel](https://vercel.com) or any Node 18+ host; add the same environment variables in the project settings.
